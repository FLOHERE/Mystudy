- count.py
	- 10부터 0까지 -1 씩 빼면서 카운트 세기
	- 1초 간격으로 출력문 실행
	- 반복문 끝나면 "발사!" 출력
```python
import time

for i in range(10,0,-1):
	print(i)
	time.sleep(1)
print("발사!")
```
