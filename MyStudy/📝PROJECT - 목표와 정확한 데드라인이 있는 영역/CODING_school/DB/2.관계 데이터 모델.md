- 릴레이션(relation)
	- 행과 열로 구성된 테이블
- 용어
	- relation : 테이블(관계 아님, 테이블이라고 읽어라)
	- realational data model : 관계 데이터 모델

	- relational algebra : 관계 대수
	- realtionship : 관계
