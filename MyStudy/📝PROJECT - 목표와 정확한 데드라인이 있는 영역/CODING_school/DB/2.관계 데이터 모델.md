## 릴레이션 [[스키마]]와 인스턴스
- 릴레이션에 사용되는 용어를 학습
	- 속성,투플, 스키마, 인스턴스
![[Pasted image 20240407224436.png]]

- 릴레이션 스키마
	- 속성(attribute)
		- 릴레이션  스키마의 열
	- 도메인(domain)
		- 속성이 가질수 있는 값의 집합
	- 차수(degree)
		- 속성의 개수

- 스키마의 표현
```SQL
릴레이션 이름(속성1 : 도메인1, 속성2: 도메인 2, ...)
		- 예) 도서(도서번호, 도서이름, 출판사, 가격)
		- 도서)(도서번호 : integer, 도서이름:char(40), 
					출판사 : char(40), 가격 : integer)
```
