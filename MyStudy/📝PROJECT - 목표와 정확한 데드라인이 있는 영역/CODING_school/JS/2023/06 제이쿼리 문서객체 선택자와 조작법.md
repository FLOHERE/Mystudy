## 06-1 제이쿼리 기본 다지기 
### 06.1.1 제이쿼리란?
- 라이브러리 언어
	- 자바스크립트로 만들어진 함수들의 집합
	-> 자바스크립트를 사용할때 불편했던점 개선 

- 호환성 문제 해결
- 쉽고 편리한 애니메이션 효과 기능 구현
- 제이쿼리 사용방법
```js
<script src="jquery-3.7.1.min.js"></script>
```

## 06-2 선택자
- HTML 요소 가져옴.
	- 선택한 요소의 디자인 속성 적용
	- 제이쿼리 사용 => 동적인 스타일 사용가능

### 06.2.1 문서 객체 모델
- DOM(Document Object Model)
	- HTML 문서 객체 구조
		- HTML 문서의 최상위객체 : `<html> = 뿌리`
		- HTML 문서의 하위객체 : `<head> <body> = 노드`
			- 노드의 종류 : 요소노드, 텍스트노드, 속성노드
	- 모든 태그 = 객체
		- 태그 = 기능 + 속성

### 06.2.2 선택자 사용하기
- 문서객체 불러오기
- 1번 방법
```js
<script>
	$(document).ready(function(){  //모든 문서가 로드 될때까지 기다렸다 실행
		$("#txt").css("color","red");
	})
</script>
```

- 2번 방법
```js
<script>
	$(function(){ //방법 1을 간결하게 표현한 것.
		$("#txt").css("color","red")
	})
</script>
```

### 06.2.3 기본선택자
- `<body>`영역에 있는 문서 객체를 선택
	- 직접 선택자
	- 인접 관계 선택자
- 문서 객체 = 요소 or 태그 

- 직접 선택자

| 종류 | 사용법 | 설명|
|---|---|---|
|전체 선택자 | $("\*\")| 모든 요소 선택
|아이디 선택자 | $("#아이디 명") | id 속성에 지정한 값을 가진 요소 선택
| 클래스 선택자 | $(".클래스 명") | class 속성에 지정한 값을 가진 요소 선택
| 요소 선택자 | $("요소명") | 지정한 요소(태그)명과 일치하는 요소들만 선택
| 그룹 선택자 | $("선택1, 선택2, 선택3,...n")| 지정된 요소들을 한번에 선택
| 종속 선택자 | \$("p.txt_1") \$("p#txt_1") | `<p>`요소 중 class의 값이 txt_1인 요소 또는 id값이 txt_1인 요소 선택|

- 인접 관계 선택자

| 종류 | 사용법 | 설명|
| ---|---|---|
| 부모 요소 선택자 | $("요소선택").parent() | 선택한 요소의 부모 요소 선택|
| 상위 요소 선택자 | $("요소선택").parents() | 선택한 요소의 상위 요소를 모두 선택|
| 가장 가까운 상위 요소 선택자 | $("요소선택").closest("div") | 선택한 요소의 상위 요소중 가장 가까운 div만 선택|
| 하위 요소 선택자 | $("요소 선택 하위요소") | 선택한 요소에 지정한 하위 요소를 선택|
| 자식 요소 선택자 | $("요소 선택>자식 요소") | 선택한 요소 기준, 자식 관계 요소만 선택|
| 자식 요소들 선택자 | $("요소선택").children()| 선택한 요소의 모든 자식 요소 선택|
| 형(이전)요소 선택자| $("요소 선택").prev() | 선택한 요소의 바로 이전 요소 선택|
| 형(이전)요소들 선택자 | $("요소 선택").prevAll()| 선택한 요소의 이전 모두 요소 선택|
| 지정 형(이전)요소들 선택자 | $("요소 선택").prevUntil("요소명")|요소 선택~요소명 이전 까지 모두 선택|
| 동생(다음) 요소 선택자| \$("요소 선택").next() \$("요소 선택 + 다음요소") | 선택한 요소의 다음 요소 선택|
| 동생(다음) 요소들 선택자 | $("요소 선택").nextAll()| 선택한 요소의 다음 요소 모두 선택|
| 지정 동생(다음)요소들 선택자 | $("요소 선택").nextUntil("h2") | 선택한 요소부터 지정한 요소의 다음 요소까지 모두 선택|
| 전체 형제 요소 선택자 | $(".box_1").sibilings() | class 값이 box_1인 요소의 형제요소 전체를 선택|

### 06.2.4 직접 선택자
- 멀리 떨어진 요소를 직접 선택할때 사용
	- 전체(\*\) 
	- 아이디(#)
	- 클래스(.)
	- 요소명(=태그)
	- 그룹(,)

#### 전체 선택자
`기본형 : $(*)`
```js
$("*").css("border","1px solid blue");
```

#### 아이디 선택자
`기본형 : $("#아이디명")`
```js
$("#tit").css("background-color","#ff0")
.css("border","2px solid #f00");
```

>[!note] 체이닝 기법
>선택한 메서드를 연속하여 사용, 한 객체에 다양한 메스드를 줄줄이 사용함.
>메서드 사용이 끝나는 부분에 ; 을 작성하여 마무리

#### 클래스 선택자
- 클래스 속성에 지정한 값을 포함하는 요소 선택
`기본형 : $(".클래스명")`
```js
$(".tit").css("background-color","#ff0")
.css("border","2px dashed #f00");
```

#### 요소명 선택자
- 지정한 요소명(Tag Name)과 일치하는 요소 모두 선택
`기본형 : $("요소명")`
```js
$("h2").css("background-color","#0ff")
.css("border","2px dashed #f00");
```

#### 그룹 선택자
- 한번에 여러개의 요소 선택
`기본형 : $(요소 선택1, 요소 선택2, 요소 선택3,...);`
```js
$("#tit3, h1").css("background-color","#0ff")
.css("border","2px dashed #f00");
```

#### 종속 선택자
- 선택한 요소의 id 또는 class 값이 일치하는 요소 선택
	- 요소명과 id(class)값 사이의 공백은 없으니 주의!@@
`기본형 : $("요소명#id값") / $("요소명.class값")`
```js
$("h1.tit").css("background-color","#0ff")
.css("border","2px dashed #f00");
```

### 06.2.5 인접 관계 선택자
- 직접 선택자 요소로 선택 후 선택한 요소와 가까이 있는 요소 선택
	- 형 = 이전
	- 동생 = 이후

#### 부모 요소 선택자
- 선택한 요소를 감싸고 있는 부모 요소 선택
`기본형 : $("요소 선택").parent();`
```js
$("#list_1").parent()
.
.
.
<ul>
	<li id="list_1">리스트</li> /*선택한 요소*/
</ul> 
/*ul 선택*/
```

#### 하위 요소 선택자
- 기준 요소 선택자의 하위 요소만 가능
`기본형 : $("기준 요소 선택1 요소 선택2")`
```js
$("#wrap h1") //id 가 wrap인 요소가 기준이됨.
```

#### 자식 요소 선택자
- 선택된 요소 기준, 지정한 자식 요소 선택.
`기본형 : $("요소 선택 > 자식 요소 선택")`
`$("요소 선택")children("자식 요소 선택")`
`$("요소 선택").children()` //모든 자식 요소 선택
```js
$("#wrap > h1").css("border","2px dashed #f00")
$("#wrap > section").children()
.css({
	"backgroound-color":"yellow",
	"border":"2px solid #f00"
})
```

#### 형(이전)/동생(다음) 요소 선택자
- 형 요소 선택자
	- 선택한 요소 기준 바로 이전 형제 요소만 선택
- 동생 요소 선택
	- 선택한 요소 기준 다음 형제 선택

`기본형 : $("요소선택").prev()`//이전
`$("요소 선택").next()`//이후
`$("요소 선택1 + 요소 선택2")`//요소 선택1 다음 바로 오는 요소선택2 만 선택

```js
$(".txt").prev()
.css(style_1);

$(".txt + p").css(style_2)//txt 바로 밑 P 태그만

$(".txt").next().next() //txt 클래스 바로 다음다음거
```

#### 전체 형(이전) / 동생(다음) 요소 선택자
- 선택한 요소 기준으로 모든 형 요소 선택 or 모든 동생 요소 선택
`기본형 : $("요소 선택").prevALl()`
`$("요소 선택").nextAll()`
```js
$(".txt").prevALl()
.css(~~)
	 
$(".txt").nextAll()
.css(~~)
```

#### 전체 형제 요소 선택자
- 선택한 요소의 모든 형제 요소 선택
`기본형 : $("요소 선택").sibilings();`
```js
$(".txt").sibilings()
.css(style_1);
```

#### 범위 제한 전체 형/동생 요소 선택자
- 선택한 요소 기준, 지정한 범위 내의 전체 형 또는 동생 요소 선택
`기본형 : $("요소 선택").prevUntil("범위 제한 요소 선택")`
`$("요소 선택").nextUntil("범위 제한 요소 선택")`
```js
$(".txt3").prevUntil(".title")
.css(style_1);

$(".txt3").nextUntil(".txt6")
.css(style_1);
```

#### 상위 요소 선택자
- 선택한 요소 기준, 모든 상위 요소 선택 
`기본형 : $("요소 선택").parents()` 
- 상위 요소 중 선택하고자 하는 요소만 선택
`$("요소 선택").parents("요소 선택")`
```js
$(".txt1").parents()
.css({
	"border":"2px dashed #00f"
})

$(".txt2").parents("div")
.css({
	"border":"2px solid #f00"
})
```

#### 가장 가까운 상위 요소 선택자
- 선택한 요소 기준, 가장 가까운 상위 요소만 선택
`기본형 : $("요소 선택").closest("요소 선택")`
```js
$(".txt1").closest("div")
```

## 06-3 제이쿼리 탐색 선택자

### 06.3.1 탐색 선택자
- 기본 선택자로 선택한 요소 중 원하는 요소를 한번 더 탐색 -> 정확히 선택 가능
	- 위치 탐색 선택자 : 배열의 인덱스를 사용해 선택
	- 속성 탐색 선택자 : 배열에 담긴 요소 중 지정된 속성과 값으로 선택
	- 콘텐츠 탐색 선택자 , 필터링 선택자 : 콘텐츠 포함 여부로 다시 선택
> 실제 배열은 아니나, 요소들의 나열에 임의로 배열 번호를 부여함.
### 06.3.2 위치 탐색 선택자
- 기본 선택자로 선택된 요소는 배열에 담김.
	- 배열 사용시 특정 요소를 좀 더 정확하게 선택


| 종류 | 사용법 | 설명|
|---|---|---|
| $("요소 선택:first") \| $("요소 선택").first() | $("li:first") \| $("li").first() |전체 `<li>`요소 중 첫번째 요소만 선택|
| \$("요소 선택:last") \| \$("요소 선택").last() | $("li:last") \| $("li").last() | 전체 `<li>`요소 중 마지막 요소만 선택|
| $("요소 선택:odd") | $("li:odd") | `<li>`요소 무리 중 홀수 인덱스 요소만 선택|
| $("요소 선택:even") | $("li: even") | `<li>`요소 무리 중 짝수 인덱스 요소만 선택|
| $("요소 선택:first-of-type")| $("li:first-of-type")| `<li>`요소 무리 중 첫번쨰 요소만 선택|
| $("요소 선택:last-of-type") | $("li:last-of-type") | `<li>`요소 무리 중 마지막 요소만 선택|
| $("요소 선택:nth-child(숫자)")| $("li:nth-child(n)") | `<li>`요소 무리 중 n번째 요소만 선택|
| $("요소 선택:nth-child(숫자n)")| $("li:nth-child(Nn)")|`<li>`요소 무리중 N의 배수번쨰 요소만 선택|
| $("요소 선택:nth-last-of-type(숫자)")| $("li:nth-last-of-type(n)")| `<li>`요소 무리 중 마지막 위치로부터 n번째에 있는 요소만 선텍|
| $("요소 선택:only-child")| $("li:only-child")| 부모 요소 내에 `<li>`요소가 한개뿐인 `<li>`요소만 선택|
| \$("요소 선택:eq(index)") \$("요소 선택").eq(index)| \$("li:eq(n)") \$("li").eq(n)| `<li>`요소 중 인덱스n이 참조하는 요소 불러옴|
| $("요소 선택:gt(index)")| $("li:gt(n)")| `<li>`요소 중 n보다 큰 인덱스가 참조하는 요소 불러옴|
| $("요소 선택:it(index)")| $("li:it(n)")| `<li>`요소 중 인덱스 n보다 작은 인덱스가 참조하는 요소를 불러옴|
| $("요소 선택").slice(index)| $("li").slice(n)| `<li>`요소 중 인덱스 n부터 참조하는 요소 불러옴|

#### first/last 선택자
- first : 선택된 요소 중 첫번째 요소만 선택
`$("요소 선택:first") / $("요소 선택").first()`
- last : 선택된 요소 중 마지막 요소만 선택
`$("요소 선택:last") / $("요소 선택").last()`
```js
$("#menu. li:first")

$("#menu li:last")
```

#### even/odd 선택자
- even : 선택한 요소 중 짝수 인덱스 요소만 선택
`$("요소 선탹:even)"`
- odd : 홀수 인덱스 요소만 선택
`$("요소 선택:odd")`
```js
$("#menu li:even")
.
.
#("mune li:odd")
```

#### eq(index)/lt(index)/gt(index) 탐색 선택자
- eq(index) : 선택한 요소 중 지정한 인덱스가 참조하는 요소만 선택
`$("요소 선택:eq(index)") | $("요소 선택").eq(index)`
- lt(index) : 지정한 인덱스 보다 작은(Less Than) 인덱스를 참조하는 요소만 선택
`$("요소 선택:lt(index)")`
- gt(index) : 지정한 인덱스 보다 큰 인덱스(Greater Than) 참조하는 요소만 선택
`$("요소 선택:gt(index)")`
```js
$("#menu li").eq(2) //아이디가 menu인 요소의 하위 요소중 2번 인덱스에 있는 li요소만 선택

$("#menu li:lt(2)")// 아이디가 menu인 요소의 하위 요소중 인덱스 번호가 2보다 작은 모든 요소

$("menu li:gt(2)")//아이디가 menu인 요소의 하위 요소중 인덱스 번호가 2보다 큰 모든 요소
```

#### first-of-type / last-of-type 선택자
- first-of-type : 요소의 무리 중 첫번째 요소만 선택
`$("요소 선택:first-of-type")`
- last-of-type : 요소의 무리 중 마지막 요소만 선택
`$("요소 선택:last-of-type")`
```js
$("li:first-of-type") // 모든<li> 태그중 가장 첫번째거
$("li:last-of-type") // 모든<li> 태그 중 가장 마지막거
```

#### nth-child(숫자n)/nth-last-of-type(숫자) 선택자
- nth-child(숫자n) : 지정한 숫자(배수)의 요소 선택
`$("요소 선택:nth-child(숫자)")` -> 숫자번째에 있는 자식 요소 선택
`$("요소 선택:nth-child(숫자n) ")` -> 숫자의 배수 번째에 있는 모든 자식요소 선택

- nth-last-of-type(숫자) : 요소 무리 중, 뒤에서 지정한 숫자번째 요소 선택
`$("요소 선택:nth-last-child(숫자)")` -> 뒤에서부터 숫자번째에 있는 자식요소 선택
```js
$("#menu1 li:nth-child(1)")//그냥 첫번째에 있는 자식요소

$("#menu1 li:nth-child(2n)")//2의 배수번째에 있는 li의 자식 요소 선택

$("#menu1 li:nth-last-child(2)") //뒤에서 두번째에 있는 요소 선탹
```

#### only-child/slice(index) 선택자
- only-child : 부모 요소에게 하나뿐인 자식 요소인 경우만 선택
`$("요소 선택:only-child")`
- slice(start index, end index) : 지정한 구간 인덱스 선택
`$("요소 선택").slice(start indx, end index)`
```js
$("li:only-child")

$("#menu1 li").slice(1,3)//1초과 3이하인 인덱스 번호를 가진 요소 모두 선택
```

| $.map() | $.map(Array, function)| 배열에 저장된 수만큼 메서드 반복 실행, 함수에서 반환된 데이터는 새 배열에 순서대로 저장. 새로 저장된 배열 객체 반환|

#### each()/$.each() 메서드
- 선택자로 선택한 요소(문서객체)는 배열에 순서대로 저장.
	- 배열에 저장된 요소를 순서대로 하나씩 선택하면서 인덱스 정보를 가져옴.
- 배열에 저장된 요소의 개수만큼 메서드 반복 실행
	- 메서드를 실행 할때마다 매개변수1 , 매개변수2에는 배열에 저장된 요소와 인덱스의 값이 배열에 오름차순으로 대입됨.
`$("요소 선택").each(function(매개변수1, 매개변수2){})`
`$.each($("요소 선택").function(매개변수1, 매개변수2){})`

`$("요소 선택").each(funciton(){$(this)})` // 매개변수 대신 this 사용(원리는 똑같음)
`$.each($("요소 선택"),function(){$(this)})`

```js
var obj=[{
	"area":"서울"
},{
	"area":"부산"
},{
	"area":"전주"
}]

$(obj).each(fucntion(i,o){ //i = index, o = object
	console.log(i + ":" + o);
})

$.each($("#menu2 li"), function(i){
	console.log(i + ":",$(this))
})
```

>[!note] 한마디 정리
> $.each 메서드는 JS의 for 문과 유사한 역할을 함 (+ 함수)

#### \$.map()/ \$.grep()메서드
- \$.map()메서드 : 배열에 저장된 데이터 수만큼 메서드 반복
	- 메서드에서 반환된 데이터 -> 새 배열에 저장
	- 배열 객체 반환
- \$.grep()메서드 : 배열에 저장된 데이터 수만큼 메서드를 반복 실행
	- 인덱스 오름차순으로 배열의 데이터 불러옴.
	- 반환값이 True면 데이터가 새 배열에 저장되고 반환함.
```js
//기본형 $.map()
$.map(Array, function(매개변수1, 매개변수2){
	return 데이터;
})

//기본형 $.grep()
$.grep(Array,function(매개변수1, 매개변수2){
	return[true | false];
})
```

- 배열의 데이터와 인덱스값이 매개변수1,2에 차례대로 대입됨.

```js
$(function(){
	var arr1 = [{
		"area":"서울", //"속성 이름":"속성값"
		"name":"무대리"
	},{
		"area":"부산",
		"name":"홍과장"
	},{
		"area":"대전",
		"name":"박사장"
	},{
		"area":"서울",
		"name":"빅마마"
	}]
	
	var arr2 = $.map(arr1, function(a,b){
		if(a.area == "서울"){ // 매개변수 a에 "area" 값이 "서울"인 것을 찾아서 대입
			return a;//새롭게 가공된 배열 생성
		}
	});
	
	var arr3 = $.grep(arr1,function(a,b){
		if(a.area == "서울"){
			return true;
		}else{
			return false;
		}
	})
})
```

#### \$in.Array()/\$.isArray()/\$.merge() 메서드
- $.inArray() : 배열에 저장된 데이터 중 지정한 데이터를 찾아 인덱스 값 반환
	- 오름차순으로, 먼저 일치하는 값을 반환
- $.isArray() : 지정한 데이터가 배열객체면 True반환, 아니면 flase
- $.merge() : 두 배열 객체를 하나로 묶음.

```js
//기본형
$.inArray(data, Array, start index)
$.isArray(object)
$.merge(Array1, Array2)
```

### 06.3.4 속성 탐색 선택자
- 선택한 요소 기준, 일치하는 속성의 포함 여부 따짐 -> 요소 선택
	- 선택한 이미지 요소 중 경로(src)가 \[imges/logo.gif]와 일치하는 이미지 요소만 선택하고자 할때 사용 합니다.

| 종류 | 사용법 | 설명 |
|---|---|---|
| $("요소 선택\[속성]") | $("li\[title]") | `<li>` 요소 중 title 속성이 포함된 요소만 선택 |
| $("요소 선택\[속성=값]") | $("li\[title=리스트]") | `<li>`요소 중 title속성값이 '리스트'인 요소만 선택 |
| $("요소 선택\[속성^=텍스트]") | $("a\[href^='http://']")| `<li>`요소 중 href 속성값이 \'http:\//'로 시작하는 요소만 선택|
| $("요소 선택\[속성\$=텍스트]") | $("a\[href\$='.com']") | `<li>`요소 중 href속성값이 '.com'으로 끝나는 요소만 선택|
| $("요소 선택\[href*=텍스트]") | $("a\[href*='easyspub']") | `<li>`요소 중 href 속성값 중에서 'easyspub'을 포함하는 요소만 선택|
| $("요소 선택:hidden") | $("li:hidden") | `<li>`요소 중 숨겨져 있는 요소만 선택합니다. | 
| $("요소 선택:visible") | $("li:visible") | `<li>`요소 중 보이는 요소만 선택|
| $(":text") | $(":text") | `<input>` 요소 중 type 속성값이 "text"인 요소만 선택합니다. |
| $(":selected") | $(":selected") | selected 속성이 적용된 요소만 선택|
| $(":checked") | $(":checked") | checked 속성이 적용된 요소만 선택|

#### 속성값에 따른 탐색 선택자
- 선택한 요소 중 지정한 속성과 일치하는 속성이 있는지, 없는지 따짐.
```js
//기본형
$("요소 선택[속성]") //속성이 포함된 요소만
$("요소 선택[속성^=값]") //명시한 값으로 시작하는 요소만
$("요소 선택[속성$=값]") //명시한 값으로 끝나는 요소만
$("요소 선택[속성*=값]") //명시한 속성값을 포함하는 요소만
$(":type 속성값") //type 속성값이 일치하는 요소만
```

#### 속성 상태에 따른 탐색 선택자
- 선택한 요소 중 속성 상태에 따라 요소 선택
```js
//기본형
$("요소 선택:[visible | hidden]") //숨겨진 상태, 보이는 상태 요소만 선택
$(":selected") //선택 상자 중 현재 선택된 옵션
$(":checked") // 체크박스 or 라디오 버튼 중 체크된 요소만 선택
```

### 06.3.5 그 외의 선택자 - 콘텐츠 탐색 선택자
- 요소 , 속성의 포함 여부에 따라 특정 요소를 선택

| 종류 | 사용법 | 설명|
|---|---|---|
| $("요소 선택:contains(텍스트)") | $("li:contains("내용2")") | `<li>`요소 중 '내용2'라는 텍스트를 포함하는 요소만 선택|
| $("요소 선택").contents() | $("p").contents() | 선택한 요소의 하위 요소 중 가장 가까운 하위 요소를 선택 | 
| $("요소 선택:has(요소명)") \| $("요소 선택").has(요소명) | $("li:has("span")") \| $("li").has("span") | `<li>`요소 중 `<span>`을 포함하는 요소만 선택|
| $("요소 선택:not(:제외요소)") \| $("요소 선택").not(:제외요소") | $("li:not(:first)") \| $("li") | `<li>`요소 중 첫번째 요소만 제외 하고 선택|
| $(요소 선택).filter(필터 요소) | $("li").filter(".list2") | `<li>`요소 중 class값이 list2인 요소만 선택|
| $(요소 선택1).find(요소 선택2) | $("li").find("strong") | `<li>`요소의 하위 요소인 `<strong>` 만 선택 | 
| $("요소 선택1").closest("요소 선택") | $("strong").closeset("div") | `<strong>`을 감싸는 상위 `<div>` 요소 중 가장 가까운 상위 요소 선택|
| end() | $("li").children("a").end() | 필터링이 되기 이전의 요소인 `<li>`가 선택 됩니다.|

#### contains()/contents()/has()/not/end() 탐색 선택자
- contains() : 선택한 요소 중 지정한 텍스트를 포함하는 요소만 선택
`$("요소 선택:contains(텍스트)")`
- contents() : 선택한 요소의 하위 요소 중 1 깊이의 텍스트와 태그 노드 선택
`$("요소 선택").contents()`
- has() : 지정한 태그를 포함하는 요소만 선택
`$("요소 선택:has(요소명)") / $("요소 선택").has("요소명")`
- not() : 지정한 요소만 제외한 채 선택'
`$("요소 선택:not(제외할 요소 선택)" / $("요소 선택").not(제외할 요소 선택)`
- end() : 필터링 되기 이전의 선택자가 적용
`$("요소 선택").탐색 선택자().end()`

#### find()/filter()
- find() : 선택한 하위 요소 중에서 find()로 필터링한 요소만 선택
`$("요소 선택").find("하위 요소 중 필터링할 요소 선택")`
- filter() : 필터링한 요소 만 선택
`$("요소 선택").filter("선택한 요소 중 필터링할 요소 선택")`

#### is()메서드
- 선택한 요소의 상태가 지정한 속성과 일치하면 True, 아니면 False
`$("요소 선택").is(":[checked | selected | visible | hideen | animated]"`

	2. selected : 선택 상자에 선택한 옵션 요소가 선택된 상태면 True, 아니면 False
	3. visible : 선택한 요소가 보이면 True, 아니면 False
	4. hidden : 선택한 요소가 숨겨져 있으면 True, 아니면 False
	5. animated : 애니메이션 동작 상태면 True, 아니면 False

| prop("상태 속성명") \| prop("상태 속성명", 새 값) | $("요소 선택").prop("상태 속성명") \| $("요소 선택").prop("상태 속성명","새 값") | 선택한 요소의 상태 속성을 가져옴. 요소를 선택하여 상태 속성에 새 값 적용|
| val() \| val("새값") | $("요소 선택").val(); \| $("요소 선택").val("새 값") | 선택한 폼 요소의 value값을 가져옴. 폼 요소 선택 -> value 속성에 새 값을 적용|
| css("속성명") \| css("속성명","새 값") | $("요소 선택").css("속성명") \| $("요소 선택").css("속성명","새 값") | 선택한 요소의 지정한 스타일 속성값을 가져옴. 요소 선택->지정한 스타일 속성에 새 값 적용|
| addClass("class 값") | $("요소 선택").addClass("class 값") | 선택한 요소의 class 속성에 새 값을 추가 |
| removeClass("class 값") | $("요소 선택").removeClass("class 값") | 선택한 요소의 class 속성에서 지정한 값만 제거|
| toggleClass("class 값") | $("요소 선택").toggleClass("class 값") | 선택한 요소의 class 속성에서 지정한 값이 있으면 제거, 속성값이 없으면 추가|
| hasClass("class 값") | $("요소 선택").hasClass("class 값") | 선택한 요소의 class 값에 지정한 클래스 값이 포함되어 있으면 True, 없으면 False |

#### html()/text() 메서드
- html() : 선택한 요소의 하위 요소를 가져와 문자열로 반환 
	- 하위요소 모두 제거 -> 새 요소로 바꿈
- text() : 선택한 요소에 포함된 전체 텍스트 가져오기
	- 선택한 하위요소 제거 -> 새 텍스트 생성
```js
//기본형
$("요소 선택").html() 
$("요소 선택").html("새 요소")
$("요소 선택").text()
$("요소 선택").text("새 텍스트")
```

#### attr()/removeAttr() 메서드
- attr()
	- 선택한 요소의 지정한 속성값 가져옴
	- 요소를 선택하여 속성값 적용
	- 요소 선택 후 지정한 여러개의 속성값 적용
```js
//기본형
$("요소 선택").attr("속성명")
$("요소 선택").attr("속성명","새 값")
$("요소 선택").attr(["속성명1" : "새 값1","속성명2":"새 값2","속성명n":"새 값n"])
```
- removeAttr() : 선택한 요소에서 기존 속성 제거
`기본형 : $("요소 선택").remove("속성")`

#### addClass() / removeClass() / toggleClass() / hasClass() 메서드
- addClass() : 선택한 요소에 클래스 생성
- removeClass() : 지정한 클래스 삭제
- toggleClass() : 지정한 클래스 없으면 생성, 있으면 삭제
- hasClass() : 지정한 클래스가 있으면 True, 없으면 False
```js
$("요소 선택").addClass("class 값")
$("요소 선택").removeClass("class 값")
$("요소 선택").toggleClass("class 값")
$("요소 선택").hasClass("class 값")
```

#### val() 메서드
- 선택한 폼 요소의 value 속성값을 가져옴
	- 새 값 적용
```js
//기본형
$("요소 선택").val()
$("요소 선택").val("새 값")
```

#### prop() 메서드
- 선택한 폼 요소(선택상자, 체크박스, 라디오 버튼)의 상태 속성값 가져오기
	- 새롭게 설정
	- 선택한 요소의 태그명, 노드 타입, 선택상자의 선택된 옵션의 인덱스 값도 알수 있음.
```js
$("요소 선택").prop("[checked | selected]")
$("요소 선택").prop("[checked | selected]",[true | false])
$("요소 선택").prop("[tagName | nodeType | selectedIndex | defaultValue]")
```
- defaultValue : 사용자가 value값을 바꿔도 초기의 속성값을 가져옴

| wrapInner() | $("요소 선택").wrapInner(새요소) | 선택한 요소의 내용을 새 요소로 각각 감싼다.|

#### before()/insertBefore()/after()/insertAfter() 메서드
- before() / insertBefore() : 선택한 요소의 이전 위치에 새 요소 생성
`기본형 : $("요소 선택").before("새 요소");`
`$("새 요소").insertBefore("요소 선택");`
- after() / insertAfter() : 선택한 요소의 다음 위치에 새 요소 생성
`기본형 : $("요소 선택").after("새 요소");`
`$("요소 선택").insertAfter("요소 선택")`
- 사용법만 다르고 기능은 같음

#### append() / appendTo() / prepend() / prependTo() 메서드
- append() / appendTo() : 선택한 요소 내 마지막 위치에 새 요소 생성 + 추가
`기본형 : $(요소 선택).append(새 요소)`
`$(새 요소).appendTo(요소 선택)`
- prepend() / prependTo() : 선택한 요소 내 앞 위치에 새 요소 생성 + 추가
`기본형 : $(요소 선택).prepend(새 요소)`
`$(새 요소).prependTo(요소 선택)`

#### clone() / empty() / remove() 메서드
- clone() : 선택한 요소 복제 
	- 인자값이 True 면 이벤트까지 복제, False면 요소만 복제 (기본값 = false)
`기본형 : $("요소 선택").clone([true | false])`
- empty() : 선택한 요소의 모든 하위요소 삭제
`기본형 : $("요소 선택").empty()`
- remove() : 선택한 요소 삭제
`기본형 : $("요소 선택").remove()`

#### replaceAll() / replaceWith() 메서드
- 선택한 요소 -> 새요소 변경
	- 선택한 모든 요소를 한거번에 바꿀때 사용함
```js 
//기본형
$("새 요소").replaceAll("요소 선택")
$("요소 선택").replaceWith("새 요소")
```

#### unwrap() / wrap() / warpAll() / wrapInner() 메서드
- unwrap() : 선택한 요소의 부모 요소 삭제
- wrap() : 선택한 요소를 각각 새 요소로 감쌈.
- wrapAll() : 선택한 요소를 한꺼번에 새 요소로 감쌈.
- wrapInner() : 모든 하위 요소 -> 새 요소로 감쌈/

```js
//기본형
$("요소 선택").unwrap()
$("요소 선택").wrap(새요소)
$("요소 선택").wrapAll(새 요소)
$("요소 선택").wrapInner(새요소)
```
